# CalcolaSconto.it 🧮

Il calcolatore di sconti online più rapido e leggero. 
Sviluppato in HTML, CSS e JavaScript puro per la massima velocità.

### 🚀 Funzionalità:
- Calcolo sconto percentuale.
- Calcolo sconto inverso (prezzo originale).
- Lista della spesa con calcolo del risparmio totale.
